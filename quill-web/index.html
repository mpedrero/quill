quill - A (very) simple blog engine

quill is an extremely simple blog engine designed and written by someone that has not got any idea of web programming (that's me) :)

Planned features include:

  • Static HTML pages
  • Entries written in Markdown
  • Customizable theme via CSS and Mustache templates (KISS :P)
  • All needed configuration via single file
  • Tag support
  • RSS/Atom

Future development includes:

  • Comments (via Disqus or custom)
  • Social integration (maybe)

For more information, you can visit Quill development blog (for now in Spanish) that, of course, is being generated by Quill ;), so you can view current status of the engine.
