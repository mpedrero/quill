Cheatsheet VIM

Modos

  • i -> Modo Insertar
  • v -> Modo Visual
  • ESC -> Modo Comandos

Comandos generales

  • :w -> Guardar archivo
  • :q -> Salir de VIM

Cortar, copiar y pegar

  • c -> Cortar selección
  • cc -> Cortar línea
  • p -> Pegar

Buscar

  • /pattern -> buscar pattern
  • n -> siguiente coincidencia
  • N -> anterior coincidencia

Pestañas

  • :tabedit {file} -> Abre file en una nueva pestaña
  • :tabs -> Listar todas las pestañas
  • gt -> Pestaña siguiente
  • gT -> Pestaña anterior
  • {i}gt -> Pestaña i
